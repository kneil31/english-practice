# Common Mistakes

Review this before every session with Ann.

## Articles
- most of the people → **most people**
- the confidence → **confidence**
- these kind → **these kinds** / **this kind**

## Grammar
- has took → **has taken**
- comes natural → **comes naturally**
- people does → **people do**
- I have working → **I have been working**
- productivity will gets → **productivity will get** / **productivity gets**
- these fun activity today is we relax → **these fun activities help us relax**

## Word Choice
- vicious slope → **slippery slope**
- conference → **confidence**
- angers → **encourages**
- pray → **repay**
- position → **possessions**
- recognization → **RECOGNITION**
- living on the table → **LEAVING on the table**
- cheat quotes → **cheat codes**
- lying behind → **lagging behind**

## Pronunciation (Say These OUT LOUD)
- laser / leaser → **LEISURE** (LEE-zhur) — mispronounced entire Session 6!
- naturate → **NURTURE**
- muzzle → **MUSCLE**
- jailbrocket → **JAILBROKE**

## Speaking Habits
- Starting sentences with "so" (50+ times in Session 6)
- "And then" chains — replacement filler when cutting "so"
- "Probably" — 15+ per session, used as verbal pause
- "Like" as filler — "like 10 hours" → just say "10 hours"
- Repeating the same point in different words (worst: 6 ways in one answer)
- Giving abstract answers without personal examples
- Hedging: "probably", "kind of", "something like"
- Burying the point — explain for 3 minutes, then state the answer
- Subject switching — we/you/they/I in the same answer
- Can't form vocabulary sentences — describes scenarios instead of using the word directly
